API Basejump: URL Shortener

User stories:
  • I can pass a URL as a parameter and I will receive a shortened URL in the JSON response.
  • When I visit that shortened URL, it will redirect me to my original link.

Example creation usage:

https://bodonkey-urlshortener.herokuapp.com/new/https://www.google.com
https://bodonkey-urlshortener.herokuapp.com/new/http://freecodecamp.com/news

Example creation output:

{ "fullength": "http://freecodecamp.com/news", "shortened": "https://bodonkey-urlshortener.herokuapp.com/NJyiElojx" }

Usage:

https://bodonkey-urlshortener.herokuapp.com/NJyiElojx

Will redirect to:

http://freecodecamp.com/news

Adding a site that is already in the database to the end of the App URL will give the database record.

https://bodonkey-urlshortener.herokuapp.com/http://freecodecamp.com/news

Will give back:

{ "fullength": "http://freecodecamp.com/news", "shortened": "https://bodonkey-urlshortener.herokuapp.com/NJyiElojx" }

While web sites are checked for form, they are NOT checked for reachability before being added to the database. This is a feature that will be added in the future.
